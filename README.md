轻量级的Java API用于音频分析，Android的兼容

关于

musicg是一个轻量级的音频分析库，用Java编写的，具有提取了高层次和水平低音频功能的目的。

该API允许开发人员提取音频功能和喜欢阅读，切割和从InputStream容易修剪操作的音频数据。它还提供了工具，数字信号处理，呈现为研发目的wavform或谱图。

该API的是Android兼容。
